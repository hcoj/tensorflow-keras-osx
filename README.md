# tensorflow-keras-osx
A [Pixi](https://github.com/prefix-dev/pixi) config to get Keras and Tensorflow working on MacOS Sequia

## Installation
1. Install Pixi ```brew install pixi```
2. Run ```pixi run clean_install_verify```
